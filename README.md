# paper figure tool

一个用于论文组图排版的桌面工具（PySide6）。

适用场景：将多张实验图快速排版为论文插图，添加编号/文本框，并导出高质量文件。

## 安装

### 1) 准备环境

- 建议 Python 3.10+
- Windows 10/11（当前项目在 Windows 环境开发）

### 2) 安装依赖

在项目根目录执行：

```bash
python -m pip install -r requirements.txt
```

依赖见 `requirements.txt`（PySide6、Pillow、reportlab、svgwrite 等）。

## 快速开始

在项目根目录执行：

```bash
python main.py
```

基础使用流程：

1. 导入图片
2. 选择网格排版（如 2×2、3×2 或自定义）
3. 添加自动编号或文本框
4. 微调位置、对齐、样式
5. 导出成目标格式

## 核心功能

- 图片导入与画布编辑
  - 拖拽/缩放/移动图片框
  - 画布尺寸与 DPI 设置（支持 mm/cm/in/px）
- 智能排版
  - 预设网格：2×2、3×2、2×4、4×2
  - 自定义行列排版
- 图片处理
  - 旋转（90°步进）、水平/垂直翻转
  - 边框设置
  - 填充模式（适配/铺满）
- 标注与文本
  - 一键自动编号（多种样式）
  - 编号样式编辑
  - 文本框添加、字体与样式设置
- 对齐与分布
  - 左/中/右、上/中/下对齐
  - 水平/垂直等距分布
- 项目管理
  - 新建、打开、保存、另存为
  - 项目文件格式：`.figproj`
- 编辑能力
  - 撤销/重做
  - 复制/剪切/粘贴、全选、删除

## 导出说明

支持导出格式：

- 位图：`JPG` / `PNG` / `TIFF`
- 文档：`PDF`
- 矢量容器：`SVG`（内部嵌入 PNG 画面）

导出要点：

- 可设置导出 DPI（72~1200）
- PNG/TIFF/PDF/SVG 支持透明背景模式
- JPG 不支持透明背景

## 说明

- 项目保存为 `.figproj` 后，可再次打开继续编辑。
- 若打开项目时提示图片缺失，请确认原始图片路径未变更或已恢复到原位置。
