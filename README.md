# paper figure tool

一个用于论文组图排版的桌面工具（PySide6）。

适用场景：将多张实验图快速排版为论文插图，添加编号/文本框，并导出高质量文件。

## 安装

### 1) 准备环境

- 建议 Python 3.10+
- Windows 10/11（当前项目在 Windows 环境开发）

### 2) 安装依赖

在项目根目录执行：

```bash
python -m pip install -r requirements.txt
```

依赖见 `requirements.txt`（PySide6、Pillow、reportlab、svgwrite 等）。

## 快速开始

在项目根目录执行：

```bash
python main.py
```

基础使用流程：

1. 导入图片
2. 选择网格排版（如 2×2、3×2 或自定义）
3. 添加自动编号或文本框
4. 微调位置、对齐、样式
5. 导出成目标格式

## 核心功能

- 图片导入与画布编辑
  - 拖拽/缩放/移动图片框
  - 画布尺寸与 DPI 设置（支持 mm/cm/in/px）
- 智能排版
  - 预设网格：2×2、3×2、2×4、4×2
  - 自定义行列排版
- 图片处理
  - 旋转（90°步进）、水平/垂直翻转
  - 边框设置
  - 填充模式（适配/铺满）
- 标注与文本
  - 一键自动编号（多种样式）
  - 编号样式编辑
  - 文本框添加、字体与样式设置
- 对齐与分布
  - 左/中/右、上/中/下对齐
  - 水平/垂直等距分布
- 项目管理
  - 新建、打开、保存、另存为
  - 项目文件格式：`.figproj`
- 编辑能力
  - 撤销/重做
  - 复制/剪切/粘贴、全选、删除

## 导出说明

支持导出格式：

- 位图：`JPG` / `PNG` / `TIFF`
- 文档：`PDF`
- 矢量容器：`SVG`（内部嵌入 PNG 画面）

导出要点：

- 可设置导出 DPI（72~1200）
- PNG/TIFF/PDF/SVG 支持透明背景模式
- JPG 不支持透明背景
- **颜色管理**：导入时自动应用 ICC 色彩管理，将内嵌 ICC profile 转换到 sRGB 色彩空间，确保显示和导出颜色一致
- **导出嵌入**：PNG/TIFF/JPG 导出时自动嵌入 sRGB ICC profile，确保在其他软件中打开时颜色正确显示
- **超大图兼容**：已启用 256MP 安全上限（`Image.MAX_IMAGE_PIXELS=256_000_000`），覆盖绝大多数科研图像，同时防止恶意/损坏文件导致 OOM
- **调色板透明度兼容**：对 `P/PA` 且 `transparency=bytes` 的图像先转 `RGBA`，避免 Pillow 透明度告警

## 颜色管理说明

本工具支持 ICC 色彩管理，确保图片在不同软件间显示颜色一致：

1. **导入处理**：自动检测图片内嵌的 ICC profile，转换到 sRGB 色彩空间
2. **灰度图处理**：保持原始灰度信息，正确处理 WhiteIsZero 标记
3. **16bit 图像**：线性映射到 8bit 显示范围，保持原始数据的相对关系
4. **调色板 PNG**：彩色调色板（P 模式）保持彩色渲染，仅灰度调色板走灰度路径
5. **导出嵌入**：所有导出格式自动嵌入 sRGB ICC profile
6. **调色板透明图处理**：`P/PA` 且 `transparency=bytes` 的输入在渲染前先统一转为 `RGBA`
7. **超大图策略**：允许超大像素图加载，并仅抑制 Pillow 的 `DecompressionBombWarning`

### 图像模式处理规则

| 模式 | 处理方式 |
|------|----------|
| `RGB` / `RGBA` | 直接保留颜色，ICC 转换后输出 |
| `P`（彩色调色板） | 检测调色板是否为灰度，彩色则转 RGB 保留颜色 |
| `P`（灰度调色板） | 走灰度渲染路径，线性映射到 8bit |
| `L` / `1` / `I` / `F` / `I;16` | 数值灰度模式，线性映射到 8bit 显示 |
| `CMYK` | 转 RGB 后输出 |
